# Daily Tasks Tracker

A simple, local-storage based daily task tracking application.

## Features

- **Dashboard**: View today's tasks and progress.
- **Task Management**: Create, edit, and delete recurring or one-time tasks.
- **Categories**: Organize tasks with custom color-coded categories.
- **History**: Calendar view to track past performance.
- **Privacy Focused**: All data is stored locally in your browser (`localStorage`).
- **Responsive Design**: Works on desktop and mobile.

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Setup

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Start tracking your tasks!

## Recent Updates

- **Timezone Fix**: Fixed an issue where tasks would show incorrect dates for users in positive timezones (e.g. UTC+6).
- **Simplified Status**: Tasks now toggle simply between "Complete" and "Not Complete".
